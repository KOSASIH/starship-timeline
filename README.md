# Starship Timeline

## Installation

### Prereqs

- Node >= v10.0.0
- Yarn >= v1.13.0
- Gatsby CLI >= v2.6.7: run `npm install -g gatsby-cli`

### Setup instructions

1. In the project directory, run `npm install` command.
2. Run `gatsby develop`.
3. Open `http://localhost:8000/`.
4. ...
5. Profit!

## Deployment

Push on the `master` branch.
